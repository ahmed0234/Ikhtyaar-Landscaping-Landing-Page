# 🌿 Landscaping Leads Engine

A modern lead generation platform built to help landscaping businesses consistently attract, qualify, and convert high-intent clients using Google Ads and a high-converting web experience.

---

## 🚀 Overview

This project is a fully functional lead generation system originally built for insurance services and later adapted for the landscaping industry.

It focuses on one core goal:

👉 Generate high-quality leads
👉 Book appointments automatically
👉 Help service businesses grow predictably

---

## 🧠 Core Idea

Instead of just running ads, this platform creates a complete funnel:

* 🎯 Targeted traffic from Google Ads
* 🧲 High-converting landing pages
* 📅 Embedded scheduling system
* 🔁 Seamless user journey from click to booking

---

## 🛠️ Tech Stack

Built with modern, scalable technologies:

* ⚛️ Next.js
* 🧩 React
* 🎨 CSS Modules
* 🌐 Responsive Design (Mobile First)
* 📅 Calendar / Booking Integration
* ☁️ Deployed via modern hosting platforms

---

## 🎨 UI & Design Philosophy

The interface is designed to feel:

* 🤍 Clean and minimal (white-based UI)
* 🔵 Trustworthy (blue color system)
* 🧊 Modern (glassmorphism & soft shadows)
* 🌌 Layered (gradients, depth, subtle 3D feel)

With strong emphasis on:

* Clear visual hierarchy
* Smooth micro-interactions
* Conversion-focused layout
